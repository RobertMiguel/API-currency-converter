O que é o Projeto ?

A afinidade do projeto é fazer um consumo de uma API com objetivo de converter moedas euro,dólar e real usando o conceito de fetch API.

Dividimos o projeto em três partes, uma para visualização de gráfico, uma para conversão e a outra para ver a cotação da moeda.
